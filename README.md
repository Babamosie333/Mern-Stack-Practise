## About
MERN Stack Learning is a static educational website designed to help organize topic-wise learning content in separate folders.  
The main page acts as a clean dashboard, while each subfolder opens independently without mixing content with other sections.

## Features
- Animated interactive homepage using Vanta.js.
- Separate subfolders for each topic.
- Clean, clickable buttons for navigation.
- Easy to expand with new folders in the future.
- GitHub Pages friendly static structure.
- Responsive and professional dark UI.
